My Project

A concise portfolio of interactive web projects built with HTML, CSS, and JavaScript: a country search tool, an image gallery with persistent likes, a todo list, a cards game, and a Pomodoro timer. Application state is stored locally using the Web Storage API.

🌍 Country Search

How does the country search function work?

The function uses a countries-array with objects for each country. When the user selects a country and clicks the "Show information" button, the corresponding data is retrieved and displayed in the result area.

What interactive features did I implement?

The user can choose between three countries in a dropdown menu. When clicking the button, information about the capital, population, best cities, and a map is displayed. If no country is selected, an error message is shown.

What did I learn most from this project?

I learned a lot about DOM manipulation, event handling, and localStorage.

🖼️Gallery

How does the gallery work?

The gallery displays images in a grid with the ability to "like" each image and open them in fullscreen. The like counter is saved in localStorage and the like button becomes highlighted.

What technologies were used?

I used HTML5 for semantic structure, CSS Grid and Flexbox for layout, and JavaScript (ES6) for DOM manipulation and event handling.

How was the like function implemented?

The like system uses localStorage to save both the number of likes and which images have been liked. CSS animations create a smooth bounce effect when you click, and the heart changes from empty (♡) to filled (♥) with red color. Formatting of large numbers (1K, 1M) prevents the UI from breaking.

📋Todo List

How does the todo list work?

The todo list allows users to add, mark as complete, and delete tasks. Tasks are saved in localStorage so they remain after page reload.

What did I learn most from?

I learned a lot about DOM manipulation, event handling and localStorage.

What would I do differently next time?

Next time I would plan the code structure better from the beginning and use a more modular approach. I would also spend more time writing comments in the code for better documentation. More testing of functionality would also be beneficial.

🎮Game

How does the cards-game work?

The game is a simple game where the user has to guess 'higher', 'lower' or 'equal' to the next card.

What technologies were used?

I used HTML5 for semantic structure, CSS for styling, and JavaScript (ES6) for game logic and DOM manipulation.

🕒Pomodoro

How does the pomodoro timer work?

The pomodoro timer is a simple timer that allows users to start, pause, and reset a work/break timer. It keeps track of the number of work sessions and breaks taken.
