# FinalProject — Portfolio

Small portfolio built with HTML, CSS and JavaScript showcasing three demos:
- Country Search & Gallery
- Cards Game (Higher / Lower)
- Pomodoro Timer

# Quick start:

```bash

npm run dev        # serves on http://localhost:8000
# or
python3 -m http.server 8000
```

# Key files:

- `index.html`, `about.html`
- `css/styles.css`
- `js/script.js`
- `projects/` (cards-game, pomodoro, project1)

Project details
- Country Search & Gallery: searchable dropdown, flag + facts display, responsive gallery with like/save state (localStorage).
- Cards Game: Higher/Lower card guessing game with score, lives, and responsive layout.
- Pomodoro Timer: Work/break timer with start/pause/reset and optional audio alerts.
